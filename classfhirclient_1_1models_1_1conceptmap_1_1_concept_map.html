Detailed Description

A statement of relationships from one set of concepts to one or more other concept systems.

Scope and Usage A concept map defines a mapping from a concept defined in one system to one or more concepts defined in other systems. Mappings are always framed within the concept of value sets - they are specific to a context of use.
