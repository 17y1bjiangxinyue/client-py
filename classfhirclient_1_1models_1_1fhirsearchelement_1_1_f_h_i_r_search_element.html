Detailed Description

This class enables pythonic creation of search URL strings.

Search elements are designed to be chained together. The first parameter instance in the chain must define resource_type, all subsequent params must have their subject set to be useful. Upon calling construct() on the last item in a chain, all instances are constructed into a URL query:

qry = Patient.where().address("Boston").gender('male').given_exact("Willis")

Then qry.construct() will create the string:

"Patient?address=Boston&gender=male&given:exact=Willis"
